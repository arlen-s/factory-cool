# FactoryCool Roles

## PM

- 整理原始 Idea，追问需求边界。
- 输出 Proposal、需求文档、验收标准。
- 将需求拆成 Frontend、Backend、QA、DevOps 任务。

## Frontend Developer

- 实现 React 大屏界面、Three.js 数字孪生视图、ECharts 图表。
- 维护前端状态、交互模拟和视觉还原。
- 更新任务状态和执行结果。

## Backend Developer

- 实现 FastAPI 数据服务、WebSocket 推送和模拟算法。
- 维护模拟数据结构、AI 决策结果和异常事件。
- 更新任务状态和执行结果。

## QA

- 按验收标准测试功能、视觉还原、实时数据刷新。
- 记录缺陷、复测结果和验收结论。

## DevOps

- 维护环境变量、Docker、CI/CD 和部署说明。
- 确保本地与容器环境可重复运行。
